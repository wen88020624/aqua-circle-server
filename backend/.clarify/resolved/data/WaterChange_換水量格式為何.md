# 釐清問題

換水記錄的換水量格式為何？

# 定位

ERM：WaterChange 實體的 water_volume 屬性（float 型別）
Feature：換水記錄管理功能中描述為「換水量(1/2、1/3、全換)」

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 使用字串格式：固定值「1/2」、「1/3」、「全換」 |
| B | 使用數值格式：float 型別，儲存實際換水量（公升） |
| C | 使用字串格式：允許任意文字描述 |
| D | 使用數值格式：儲存比例（0.5、0.33、1.0） |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- WaterChange.water_volume 屬性的資料型別
- 換水記錄管理功能的驗證規則
- 換水記錄的查詢與統計功能

# 優先級

Medium
- Medium：影響邊界條件或測試完整性

# 解決記錄

- **回答**：D - 使用數值格式：儲存比例（0.5、0.33、1.0）
- **更新的規格檔**：spec/erm.dbml, spec/features/換水記錄管理.feature
- **變更內容**：
  - 更新 ERM 中 WaterChange.water_volume 屬性的 note，明確說明儲存比例（0.5 表示 1/2、0.33 表示 1/3、1.0 表示全換），並添加約束：必須 > 0 且 <= 1
  - 更新 Feature 檔案中的換水量描述和 Example，將「1/2」改為數值格式「0.5」
