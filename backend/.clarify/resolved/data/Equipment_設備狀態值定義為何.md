# 釐清問題

設備的狀態值定義為何？

# 定位

ERM：Equipment 實體的 status 屬性
Feature：設備管理功能中 Rule "狀態不可為空"

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 使用 ERM 定義：運作中、閒置、檢修、汰換 |
| B | 使用 Feature 定義：使用中、閒置、賣掉、丟棄 |
| C | 合併兩者：運作中、使用中、閒置、檢修、汰換、賣掉、丟棄 |
| D | 其他組合（請說明） |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- Equipment.status 屬性的資料型別與約束
- 設備管理功能的狀態驗證規則
- 所有涉及設備狀態查詢的功能

# 優先級

High
- High：阻礙核心功能定義或資料建模

---
# 解決記錄

- **回答**：B - 使用 Feature 定義：使用中、閒置、賣掉、丟棄
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：更新 Equipment.status 屬性的 note，將狀態值從「運作中、閒置、檢修、汰換」改為「使用中、閒置、賣掉、丟棄」，與 Feature 定義一致
