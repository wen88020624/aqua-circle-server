# Aqua-Circle (魚缸管理系統) 規格

## 概述
本系統提供魚缸管理功能，使用者可以建立和管理魚缸，並在魚缸中管理生物、設備、耗材、換水記錄、餵食記錄、下藥記錄與水質檢測記錄。

## 資料實體

### 魚缸 (Aquarium)
魚缸是系統中的主要容器，每個魚缸具有以下屬性：
- **名稱**：魚缸的名稱
- **魚缸大小**：魚缸的尺寸，包含長度、寬度、高度（單位：公分）
- **狀態**：魚缸的狀態（開缸、穩定、治療、閒置）
- **開缸日期**：魚缸開始使用的日期
- **備註**：魚缸的備註說明

### 設備 (Equipment)
設備是魚缸使用的器材，每個設備具有以下屬性：
- **名稱**：設備的名稱
- **tag**：設備的標籤（燈具、過濾器）
- **購買日期**：設備的購買日期
- **金額**：設備的購買金額
- **狀態**：設備的狀態（使用中、閒置、賣掉、丟棄）
- **備註**：設備的備註說明
- **所屬魚缸**：設備所屬的魚缸

### 生物 (Creature)
生物是養殖在魚缸中的生物，每個生物具有以下屬性：
- **名稱**：生物的名稱
- **tag**：生物的標籤（上層魚、底棲魚、異形、古代魚、蝦...）
- **購買日期**：生物的購買日期
- **金額**：生物的購買金額
- **健康程度**：生物的健康程度（過胖、過瘦、正常、生病、精神異常、死亡）
- **性別**：生物的性別（雄、雌）
- **長度**：生物的長度（單位：公分）
- **備註**：生物的備註說明
- **所屬魚缸**：生物所屬的魚缸

### 換水記錄 (WaterChange)
換水記錄是魚缸的維護記錄，每個換水記錄具有以下屬性：
- **日期**：換水的日期
- **換水量**：換水的量（1/2、1/3、全換）
- **所屬魚缸**：換水記錄所屬的魚缸
- **備註**：換水記錄的備註說明

### 耗材 (Consumable)
耗材是魚缸使用的消耗性物品，透過 tag 區分為藥品、濾材、飼料，每個耗材具有以下屬性：
- **名稱**：耗材的名稱
- **tag**：耗材的標籤（藥品、濾材、飼料）
- **數量**：耗材的數量
- **狀態**：耗材的狀態（使用中、用完、丟棄）
- **金額**：耗材的購買金額
- **備註**：耗材的備註說明
- **購買日期**：耗材的購買日期
- **到期日期**：耗材的到期日期
- **所屬魚缸**：耗材所屬的魚缸

### 餵食記錄 (FeedingRecord)
餵食記錄是魚缸的餵食記錄，每個餵食記錄具有以下屬性：
- **日期**：餵食的日期
- **所屬魚缸**：餵食記錄所屬的魚缸
- **使用的耗材**：餵食使用的耗材（tag 為「飼料」的耗材）
- **備註**：餵食記錄的備註說明

### 下藥記錄 (MedicationRecord)
下藥記錄是魚缸的用藥記錄，每個下藥記錄具有以下屬性：
- **下藥名稱**：下藥的名稱
- **tag**：下藥記錄的標籤（抗生素、病毒藥）
- **下藥的量**：下藥的用量（單位：mg/L）
- **日期**：下藥的日期
- **所屬魚缸**：下藥記錄所屬的魚缸
- **備註**：下藥記錄的備註說明

### 水質檢測記錄 (WaterQualityRecord)
水質檢測記錄是魚缸的水質檢測記錄，每個水質檢測記錄具有以下屬性：
- **檢測種類**：檢測的種類（NH3+NH4、PH、NO3、NO2）
- **檢測日期**：檢測的日期
- **數值**：檢測的數值
- **所屬魚缸**：水質檢測記錄所屬的魚缸
- **備註**：水質檢測記錄的備註說明

## 功能需求

### 1. 魚缸管理

#### 1.1 建立魚缸
使用者可以建立新的魚缸，並設定魚缸的名稱、魚缸大小（長度、寬度、高度）、狀態（開缸、穩定、治療、閒置）、開缸日期、備註。魚缸的長度、寬度、高度皆必須 > 0。

#### 1.2 查詢魚缸
使用者可以查詢所有魚缸，或查詢特定魚缸的詳細資訊。

#### 1.3 更新魚缸
使用者可以更新魚缸的名稱、魚缸大小（長度、寬度、高度）、狀態、開缸日期、備註。

#### 1.4 刪除魚缸
使用者可以刪除魚缸。

### 2. 生物管理

#### 2.1 新增生物
使用者可以在魚缸中新增生物，並設定生物的名稱、tag（上層魚、底棲魚、異形、古代魚、蝦...）、購買日期、金額、健康程度（過胖、過瘦、正常、生病、精神異常、死亡）、性別（雄、雌）、長度（公分）、備註、所屬魚缸。生物的名稱和 tag 不可為空，金額和長度必須 >= 0。

#### 2.2 查詢生物
使用者可以查詢所有生物，或查詢特定魚缸中的生物。

#### 2.3 更新生物
使用者可以更新生物的名稱、tag、購買日期、金額、健康程度、性別、長度、備註、所屬魚缸。更新時生物必須存在於系統中。

#### 2.4 刪除生物
使用者可以刪除生物。

### 3. 設備管理

#### 3.1 新增設備
使用者可以在魚缸中新增設備，並設定設備的名稱、tag（燈具、過濾器）、狀態（使用中、閒置、賣掉、丟棄）、金額、備註、購買日期、所屬魚缸。設備的名稱、tag 和狀態不可為空，金額必須 >= 0。

#### 3.2 查詢設備
使用者可以查詢所有設備，或查詢特定魚缸中的設備。

#### 3.3 更新設備
使用者可以更新設備的名稱、tag、購買日期、金額、狀態、備註、所屬魚缸。更新時設備必須存在於系統中。

#### 3.4 刪除設備
使用者可以刪除設備。

### 4. 換水記錄管理

#### 4.1 新增換水記錄
使用者可以在魚缸中新增換水記錄，並設定換水記錄的日期、換水量（1/2、1/3、全換）、備註、所屬魚缸。日期、換水量和所屬魚缸不可為空。

#### 4.2 查詢換水記錄
使用者可以查詢所有換水記錄，或查詢特定魚缸中的換水記錄。

#### 4.3 更新換水記錄
使用者可以更新換水記錄的日期、換水量、備註、所屬魚缸。

#### 4.4 刪除換水記錄
使用者可以刪除換水記錄。

### 5. 耗材管理

#### 5.1 新增耗材
使用者可以新增耗材，並設定耗材的名稱、tag（藥品、濾材、飼料）、數量、狀態（使用中、用完、丟棄）、金額、備註、購買日期、到期日期、所屬魚缸。新增時數量必須 > 0，tag 不可為空，金額必須 >= 0。當數量 > 0 時，狀態自動設為「使用中」；當數量 = 0 時，狀態自動設為「用完」。

#### 5.2 查詢耗材
使用者可以查詢所有耗材，或查詢特定 tag 的耗材（藥品、濾材、飼料），或查詢特定耗材的詳細資訊。

#### 5.3 更新耗材
使用者可以更新耗材的名稱、tag、購買日期、到期日期、金額、狀態、備註、數量、所屬魚缸。更新時耗材必須存在於系統中，tag 不可為空，數量必須 >= 0，金額必須 >= 0。當數量更新為 0 時，狀態自動更新為「用完」。

#### 5.4 刪除耗材
使用者可以刪除耗材。

### 6. 餵食記錄管理

#### 6.1 新增餵食記錄
使用者可以在魚缸中新增餵食記錄，並設定餵食記錄的日期、選擇 tag 為「飼料」的耗材、備註、所屬魚缸。新增時只能選擇 tag 為「飼料」的耗材。若使用的耗材已過期，系統會顯示警告訊息但仍允許新增。

#### 6.2 查詢餵食記錄
使用者可以查詢所有餵食記錄，或查詢特定魚缸中的餵食記錄。

#### 6.3 更新餵食記錄
使用者可以更新餵食記錄的日期、使用的耗材（tag 為「飼料」）、備註、所屬魚缸。

#### 6.4 刪除餵食記錄
使用者可以刪除餵食記錄。

### 7. 下藥記錄管理

#### 7.1 新增下藥記錄
使用者可以在魚缸中新增下藥記錄，並設定下藥記錄的下藥名稱、tag（抗生素、病毒藥）、下藥的量（mg/L）、日期、備註、所屬魚缸。下藥名稱、tag、下藥的量和所屬魚缸不可為空。

#### 7.2 查詢下藥記錄
使用者可以查詢所有下藥記錄，或查詢特定魚缸中的下藥記錄。

#### 7.3 更新下藥記錄
使用者可以更新下藥記錄的下藥名稱、tag、下藥的量、日期、備註、所屬魚缸。

#### 7.4 刪除下藥記錄
使用者可以刪除下藥記錄。

### 8. 水質檢測記錄管理

#### 8.1 新增水質檢測記錄
使用者可以在魚缸中新增水質檢測記錄，並設定水質檢測記錄的檢測種類（NH3+NH4、PH、NO3、NO2）、檢測日期、數值、備註、所屬魚缸。檢測種類、檢測日期、數值和所屬魚缸不可為空。

#### 8.2 查詢水質檢測記錄
使用者可以查詢所有水質檢測記錄，或查詢特定魚缸中的水質檢測記錄。

#### 8.3 更新水質檢測記錄
使用者可以更新水質檢測記錄的檢測種類（NH3+NH4、PH、NO3、NO2）、檢測日期、數值、備註、所屬魚缸。

#### 8.4 刪除水質檢測記錄
使用者可以刪除水質檢測記錄。
