# 釐清問題

刪除魚缸時是否需要檢查相關聯的資料？

# 定位

Feature：指出魚缸管理功能的刪除規則缺乏前置條件定義

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 必須先刪除所有相關資料才能刪除魚缸 |
| B | 刪除魚缸時自動級聯刪除所有相關資料 |
| C | 僅檢查魚隻和設備，其他記錄保留 |
| D | 不檢查，直接刪除但保留相關資料 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響魚缸刪除功能的業務邏輯、資料完整性檢查，以及相關的魚隻、設備、記錄等所有關聯實體的處理

# 優先級

High
- High：阻礙核心功能定義或資料建模
