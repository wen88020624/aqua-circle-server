# 釐清問題

耗材管理功能中數量與狀態的自動更新規則是否完整？

# 定位

Feature：耗材管理功能中：
- Rule: 新增耗材，數量 > 0，狀態自動更新為「使用中」
- Rule: 新增耗材，數量 = 0，狀態自動更新為「用完」
- Rule: 更新耗材的數量為 0 時，狀態自動更新為「用完」

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 規則完整，無需補充 |
| B | 需要補充：更新耗材數量從 0 變為 > 0 時，狀態應自動更新為「使用中」 |
| C | 需要補充：更新耗材數量從 > 0 變為 0 時，狀態應自動更新為「用完」（已有） |
| D | 需要補充其他狀態轉換規則 |
| Short | 提供其他簡短答案（<=5 字） |

# 影響範圍

- 耗材管理功能的狀態自動更新邏輯
- 耗材狀態轉換的完整性
- 測試案例的設計

# 優先級

Low
- Low：優化或細節調整
