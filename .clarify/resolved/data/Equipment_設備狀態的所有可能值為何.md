# 釐清問題

設備狀態的所有可能值為何？

# 定位

ERM：指出 Equipment 實體的 status 屬性定義不完整

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 正常、故障、維修中、報廢 |
| B | 啟用、停用、維護、損壞 |
| C | 運作中、閒置、檢修、汰換 |
| D | 良好、警告、異常、停機 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響 Equipment 實體的狀態管理、設備生命週期追蹤，以及設備管理功能的狀態篩選與維護提醒

# 優先級

High
- High：阻礙核心功能定義或資料建模
