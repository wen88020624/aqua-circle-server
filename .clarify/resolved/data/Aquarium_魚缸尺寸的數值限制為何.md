# 釐清問題

魚缸尺寸的數值限制為何？

# 定位

ERM：指出 Aquarium 實體的 length、width、height 屬性缺乏邊界條件定義

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 長寬高皆須 > 0，最大 1000 公分 |
| B | 長寬高皆須 >= 1，最大 500 公分 |
| C | 長寬高皆須 > 0，無最大限制 |
| D | 長寬須 >= 10，高須 >= 5，最大 2000 公分 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響魚缸建立與更新功能的輸入驗證、資料庫約束設定，以及容積計算的邊界處理

# 優先級

High
- High：阻礙核心功能定義或資料建模

---
# 解決記錄

- **回答**：D - 長寬須大於等於 10，高須大於等於 5，最大 2000 公分
- **更新的規格檔**：spec/erm.dbml
- **變更內容**：在 Aquarium 實體的 length、width、height 屬性 note 中新增數值範圍限制
