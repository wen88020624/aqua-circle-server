# 定位

ERM：指出 Fish 實體的 status 屬性定義不完整

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 健康、生病、死亡、隔離 |
| B | 正常、異常、治療中、已逝 |
| C | 活著、患病、康復中、死亡 |
| D | 良好、觀察、醫治、移除 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響 Fish 實體的生命狀態管理、健康追蹤，以及魚隻管理功能的狀態篩選與照護提醒
