# 釐清問題

下藥記錄新增時是否需要檢查藥品到期日？

# 定位

Feature：指出下藥記錄管理功能的新增規則缺乏前置條件定義

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 必須檢查，過期藥品禁止使用 |
| B | 過期藥品給予警告但仍可使用 |
| C | 僅檢查藥品是否存在 |
| D | 不檢查到期日 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響下藥記錄新增功能的安全性檢查、藥品庫存管理，以及魚隻用藥安全控制

# 優先級

High
- High：阻礙核心功能定義或資料建模

---
# 解決記錄

- **回答**：B - 過期藥品給予警告但仍可使用
- **更新的規格檔**：spec/features/下藥記錄管理.feature
- **變更內容**：新增藥品到期日檢查規則，定義過期藥品會顯示警告但仍可使用，正常藥品直接成功
