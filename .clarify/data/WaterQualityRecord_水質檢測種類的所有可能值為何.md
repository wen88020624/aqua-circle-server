# 釐清問題

水質檢測種類的所有可能值為何？

# 定位

ERM：指出 WaterQualityRecord 實體的 test_type 屬性僅列出四種（NH3+NH4、PH、NO3、NO2）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | NH3+NH4、PH、NO3、NO2 |
| B | NH3+NH4、PH、NO3、NO2、溫度、鹽度 |
| C | NH3+NH4、PH、NO3、NO2、GH、KH、溫度 |
| D | 上述全部加上溶氧量、導電度 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響水質檢測記錄管理功能的檢測項目選擇、水質報表生成，以及水質警示系統的完整性

# 優先級

Medium
- Medium：影響邊界條件或測試完整性
