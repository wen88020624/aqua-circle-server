# 釐清問題

濾材操作類型的所有可能值為何？

# 定位

ERM：指出 FilterMedia 實體的 operation_type 屬性僅列出部分值（清洗或更換）

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 清洗、更換 |
| B | 清洗、更換、新增、移除 |
| C | 清洗、部分更換、全部更換 |
| D | 清洗、更換、檢查、調整 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響濾材記錄管理功能的操作類型選擇、維護歷史統計，以及濾材生命週期追蹤

# 優先級

Medium
- Medium：影響邊界條件或測試完整性
