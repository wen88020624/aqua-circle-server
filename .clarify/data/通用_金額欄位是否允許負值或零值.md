# 釐清問題

金額欄位是否允許負值或零值？

# 定位

ERM：指出 Equipment、Fish、Feed、Medicine 實體的 amount 屬性缺乏數值範圍限制

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 必須 > 0，不允許負值或零值 |
| B | 必須 >= 0，允許零值但不允許負值 |
| C | 允許負值（表示退款或折扣） |
| D | 依實體類型而異 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響所有包含 amount 屬性的實體建立與更新功能，以及財務計算與報表統計的正確性
