# 釐清問題

魚缸狀態的所有可能值為何？

# 定位

ERM：指出 Aquarium 實體的 status 屬性定義不完整

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 運行中、維護中、已關閉 |
| B | 正常、警告、異常、停用 |
| C | 啟用、暫停、維護、終止 |
| D | 開缸、穩定、問題、關缸 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響 Aquarium 實體的狀態管理、狀態轉換邏輯、查詢過濾條件，以及相關的魚隻管理、設備管理等功能中的前置條件檢查
