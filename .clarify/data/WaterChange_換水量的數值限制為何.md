# 釐清問題

換水量的數值限制為何？

# 定位

ERM：指出 WaterChange 實體的 water_volume 屬性缺乏邊界條件定義

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 必須 > 0，且不能超過魚缸容積 |
| B | 必須 >= 1，最大限制由魚缸決定 |
| C | 必須 > 0，無最大限制 |
| D | 允許 0（表示清洗但不換水） |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響換水記錄建立功能的輸入驗證、魚缸水質管理邏輯，以及換水歷史統計的準確性
