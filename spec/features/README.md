# Feature 分類說明

本目錄依照「前端頁面/使用者任務」分組，而非單純依 API 動作分組。

## 分組

- `aquarium-core/`
  - `魚缸管理.feature`
- `organism-management/`
  - `生物管理.feature`
- `resource-management/`
  - `設備管理.feature`
  - `耗材管理.feature`
- `record-management/`
  - `餵食記錄管理.feature`
  - `換水記錄管理.feature`
  - `水質檢測記錄管理.feature`
  - `下藥記錄管理.feature`
